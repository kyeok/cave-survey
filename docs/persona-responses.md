### 패턴 관찰

**1. CE-CW 갭**
- 민수(IC), 수연(대학원생): CW > CE — 개인 본업에는 AI를 적극 쓰지만, 조직 맥락 구성에는 미흡
- 재영(리더), 성호(경영진): CE > CW — 관리·운영 자동화에 집중, 본업에는 상대적으로 미활용
- 정우(PO), 도현(스타트업): CE ≈ CW — 균형 잡힌 통합 활용

**2. 주체 방향 × 자율성**
- 자발 주도 + P1~P2 (민수, 수연) → 자율성 체감 높음 (D4=4)
- 시스템 주도 + P2 (재영, 성호) → 감시 체감도 존재 (D5=3)
- 시스템 주도 + P3 (도현) → 투명성이 높아 자율성도 높음 → **투명성이 감시를 완화**하는 사례

**3. 투명성 → 자율성 가설**
- 도현(투명성 5.0, 자율성 5.0): 가설 지지 — AI로 모든 정보 투명, 자율성 최고
- 태준(투명성 1.67, 자율성 5.0): 반례? — 교수 직위 자체가 자율적이라 투명성과 무관
- 은지(투명성 2.33, 자율성 2.56): 가설 방향 일치 — 투명성↓ 자율성↓
- 성호(투명성 3.0, 자율성 4.89): 경영진은 직위로 인한 자율성 → 투명성과 독립적

→ **가설은 구성원 레벨에서 더 유효하고, 리더/교수/경영진은 직위 자체의 자율성이 있어 교란 변수가 됨.**
