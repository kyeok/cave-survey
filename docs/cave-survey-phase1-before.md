## Part A-1. AI Ops Pattern — Context Engineering (객관식 6문항)

> **Context Engineering**: AI로 맥락을 구성하는 활동 — 정보를 수집·정리·연결·공유하여 팀의 판단 품질을 높이는 것
>
> 각 영역에서 **현재** AI가 어떤 수준으로 관여하고 있는지 가장 가까운 것을 골라주세요.
>
> **(N/A)** 이 활동은 내 역할에 해당하지 않는다 — **(0)** AI 미사용 — **(1)** AI 보조 — **(2)** AI 협업 — **(3)** AI 위임 — **(4)** AI 자율
>
> ⚡ **(1) 이상을 선택한 경우**, 이 AI 활용이 **주로 누구의 주도로** 이루어지는지도 함께 골라주세요.
>
> **구성원이 응답할 때:**
> **(a)** 내가 스스로 선택하여 사용
> **(b)** 리더/관리자가 권장하여 사용
> **(c)** 조직 차원에서 시스템으로 도입되어 사용
>
> **리더가 응답할 때:**
> **(a)** 내가 개인적으로 선택하여 사용
> **(b)** 내가 리더로서 팀에 도입을 주도
> **(c)** 상위 조직/전사 차원에서 시스템으로 도입

## 결과 산출 로직

### 1. AI Ops Pattern 판정

```
CE Pattern = round(mean(A1~A6 중 N/A 제외))
CW Pattern = round(mean(A7~A9 중 N/A 제외))
종합 Pattern = round(mean(A1~A9 중 N/A 제외))
```

> N/A 응답은 평균 계산에서 제외. CW 전체가 N/A이면 CW Pattern = "해당 없음"으로 표시.

| 평균 범위 | 판정 | 의미 |
|----------|------|------|
| 0.0 ~ 0.4 | **P0** | AI 미사용 |
| 0.5 ~ 1.4 | **P1** | AI 보조 |
| 1.5 ~ 2.4 | **P2** | AI 협업 |
| 2.5 ~ 3.4 | **P3** | AI 위임 |
| 3.5 ~ 4.0 | **P4** | AI 자율 |

### 2. 주체 방향 분석

```
Bottom-up 비율 = (a 응답 수) / (P1 이상 & N/A 아닌 문항 수)
```

| 비율 | 유형 | 해석 |
|:---:|------|------|
| ≥ 70% | **자발 주도형** | 자율성 강화 기대 |
| 40~69% | **혼합형** | 전환기 |
| ≤ 39% | **시스템 주도형** | 감시 체감 가능성 주의 |

**리더와 구성원의 주체 방향은 별도로 해석**:
- 리더의 (b) = "리더 주도 도입" → 능동적 의사결정
- 구성원의 (b) = "리더가 시켜서 사용" → 수동적 채택

### 3. AI 활용 행동 패턴 점수

```
CE 행동 = mean(B1, B2, B3, B4)    → 1.0 ~ 5.0
CW 행동:
  위임 수준   = B5              (높을수록 AI 결과를 신뢰)
  검증 능력   = B6              (높을수록 AI 오류를 스스로 포착)
  최적화 행동 = B7              (높을수록 AI를 반복 활용)
  역량 확장   = B8              (높을수록 AI로 새로운 영역 수행)
```

### 4. 자율성 점수

```
자율성 기준선:
  의사결정 자율성 = mean(C1, C2, C3)    → 1.0 ~ 5.0
  업무방식 자율성 = mean(C4, C5, C6)    → 1.0 ~ 5.0
  일정 자율성     = mean(C7, C8, C9)    → 1.0 ~ 5.0
  기준선 종합     = mean(C1~C9)

AI 기여 자율성:
  AI 기여분 = mean(C10, C11, C12)       → 1.0 ~ 5.0
```

> **교차 해석**: 기준선이 높고 AI 기여분이 낮으면 → 직위 자체의 자율성 (교수, 경영진 유형)
> 기준선이 보통이고 AI 기여분이 높으면 → AI가 자율성을 실질적으로 증가시킨 경우

### 5. 정보 투명성 + AI 인식

```
정보 투명성 = mean(D1, D2, D3)    → 1.0 ~ 5.0
AI 납득성   = D5
감시 체감   = mean(D6, D7, D8)    → 1.0 ~ 5.0  (3문항 평균)
```

**가설 검증 포인트**:
- 투명성↑ + AI 기여 자율성↑ → 가설 지지
- 투명성↑ + AI 기여 자율성→ → 정보는 공개되었으나 전환 미발생
- 투명성↑ + 감시 체감↑ → 투명성의 역설

**감시 체감 3차원**:
- D6 = 가시성 증가 (업무가 보임)
- D7 = 평가 연결 우려 (데이터가 평가에 쓰일까)
- D8 = 행동 위축 (기록이 남는 것에 대한 부담)

**위임 전환 준비도**:
- D10 > D9 → 위임 전환이 순조로움
- D9 > D10 → 위임 허들이 존재
