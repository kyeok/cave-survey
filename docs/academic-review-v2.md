## 남아 있는 약점

1. Part D가 5개 하위 척도로 복잡 (분리 검토 가능)
2. 역방향 문항 부재 (n=5에서는 불필요)
3. B5~B8 개별 지표 사용의 정당화 필요
4. 심리적 안전감 2문항은 원본 7문항의 적응 버전으로 명시

## 학술 발표 남은 과제

- Phase 1 실시 (n=5) + 피드백 수집
- Phase 3 실시 + Before-After 비교
- "탐색적 사례연구"로 프레이밍 (Yin 2018)
- n=5 한계 명시 + 후속 타당화 제안
